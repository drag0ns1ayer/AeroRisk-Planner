# v2.5 运行速查

所有命令都从项目根目录执行：

```powershell
cd C:\Users\20340\Desktop\project1
```

推荐使用统一脚本，不需要记完整 Python 参数：

```powershell
.\v25\scripts\v25-run.cmd help
```

## 常用流程

| 目的 | 命令 |
|---|---|
| 检查环境和单元测试 | `.\v25\scripts\v25-run.cmd test` |
| 运行 A* 扰动演示 | `.\v25\scripts\v25-run.cmd astar` |
| 新协议冒烟训练 | `.\v25\scripts\v25-run.cmd train-smoke` |
| 新协议主训练 | `.\v25\scripts\v25-run.cmd train` |
| 新协议长训练 | `.\v25\scripts\v25-run.cmd train-long` |
| 评估模型 | `.\v25\scripts\v25-run.cmd eval <模型路径>` |
| 公平比较 A* 与 A*+RL | `.\v25\scripts\v25-run.cmd compare <模型路径>` |

模型路径不需要写 `.zip`。

示例：

```powershell
.\v25\scripts\v25-run.cmd eval v25/artifacts/models/ppo_v25_trueworld_main_best/best_model
.\v25\scripts\v25-run.cmd compare v25/artifacts/models/ppo_v25_trueworld_main_best/best_model
```

## 推荐顺序

```text
test
  ↓
astar
  ↓
train-smoke
  ↓
train
  ↓
eval
  ↓
compare
```

确认主训练仍未收敛时，再运行 `train-long`。

## 输出位置

| 内容 | 目录 |
|---|---|
| 模型、评估结果 | `v25/artifacts/models/` |
| TensorBoard 与训练日志 | `v25/artifacts/logs/` |
| A* 扰动演示 | `results/single_astar_disruptive/` |
| A* 与 RL 公平比较 | `results/compare_astar_rl_disruptive/` |

## 重要说明

- 当前真世界协议使用 `41` 维观测；旧 `31` 维 v2.5 模型不能继续训练或参与比较。
- A* 与 A*+RL 比较时使用相同任务、相同随机 seed、相同真实风场和相同动力学。
- 新训练必须从头开始。脚本中的训练命令已自动包含 `--from-scratch`。
- 显存或内存不足时，在 `v25-run.ps1` 中降低 `--n-envs`。

需要完整参数时：

```powershell
python v25/train_rl_disruptive.py --help
python v25/experiments/compare_astar_rl_disruptive.py --help
```
